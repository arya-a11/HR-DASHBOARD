# 📊 HR Dashboard Project

## 🚀 Project Overview
This project focuses on building an **Interactive HR Dashboard** to analyze employee data and generate meaningful insights for decision-making. The dashboard transforms raw HR data into visually appealing and easy-to-understand reports.

It helps organizations monitor employee performance, track promotions, analyze compensation, and understand workforce distribution.

---

## 🎯 Objectives
- Analyze employee performance across departments  
- Track promotions and bonus distribution  
- Understand workforce distribution (country & department)  
- Evaluate employee training participation  
- Enable data-driven HR decision-making  

---

## 📁 Dataset Description
The dataset contains **200 employee records** across **10 departments** and **10 countries**.

### 🔹 Key Columns:
- Employee ID  
- Department  
- Country  
- Gender  
- Employment Type  
- Performance Rating  
- Education Level  
- Promotions Received  
- Bonus Percentage  
- Training Sessions  

---

## 📊 Dashboard Features

### 🔹 KPI Metrics
- Total Employees  
- Total Departments  
- Total Countries  

### 🔹 Visualizations
- Performance Ratings Analysis  
- Work Mode Distribution  
- Promotion Analysis  
- Country-wise Employee Distribution  
- Education Level Analysis  
- Bonus Distribution  
- Training Sessions Tracking  

### 🔹 Filters
- Gender  
- Performance Level  
- Promotion Status  

---

## 🔍 Key Insights
- Majority of employees are **Full-Time**  
- Performance is balanced across departments  
- Promotions are higher in **Sales & Engineering**  
- Training participation is increasing  
- Bonus distribution varies by department  
- Workforce is globally distributed
